# Archivo del Carnaval del Campo de Gibraltar

Web estática para GitHub Pages conectada a Google Sheets.

## Importante sobre autores, componentes y premios
Usa punto y coma `;` para separar varios valores dentro de una misma celda:

- Autores: `Juan Pérez; María López; Antonio García`
- Componentes: `Componente 1; Componente 2; Componente 3`
- Premios / Palmarés: `1º Premio Comparsa; Mejor Pasodoble; Mejor Tipo`
- Fotos: `https://...foto1.jpg; https://...foto2.jpg`
- Vídeos: `https://youtube.com/...; https://...`

No uses comas como separador principal porque las comas también forman parte natural de nombres, descripciones y textos.
