# Archivo del Carnaval del Campo de Gibraltar - v3

Sube todos estos archivos a GitHub manteniendo carpetas.

## Importante
Autores, componentes y premios deben separarse con punto y coma `;`, no con comas.

Ejemplo autores:
`Juan Pérez; Antonio García; María López`

Ejemplo premios:
`1º Premio Chirigotas; Mejor Tipo; Premio Pasodoble`

## Por qué
La coma se usa internamente en CSV. El punto y coma permite que una misma agrupación tenga muchos autores, componentes o premios sin duplicarse.

## Panel admin
`admin.html` tiene un panel local básico con código `CARNAVAL2026`. No es seguridad real en GitHub Pages. Para que usuarios se registren y solo tú edites de verdad hace falta backend: Supabase, Firebase o Netlify Identity.
